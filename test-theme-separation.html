Theme Separation Test

Test Instructions:

This test demonstrates how input and output messages can have different themes:

  • Input messages (user): Use themes suitable for user input (default, transparent, light, dark)
  • Output messages (assistant): Use themes suitable for assistant output (glass, blue, green, purple, etc.)
  • Click the theme button on each message to see the available themes for that message type
  • Notice how the dropdown shows different themes based on message type

Theme Configuration:

Input Themes: default, transparent, dark, light

Output Themes: glass, blue, green, purple, orange, pink, gradient, neon

Universal Themes: default, transparent, dark
